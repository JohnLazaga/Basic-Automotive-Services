The "Reset Password" button — easy guide

A new button lets you give any worker a brand-new secret password with one click. This guide shows how to switch it on (one time) and how to use it every day.

What it does (in plain words)

In the app there is a page called Accounts & Roles. Next to each worker there is now a 🎲 Reset button.

When you click it, the app:

  • Makes the worker a brand-new random password (like Motor-4182-Pusa-7731).
  • Shows it to you one time so you can copy it.
  • Their old password stops working right away.
  • Saves nothing. The password is not kept anywhere — that keeps it safe.
You never need to remember a worker's password. If it's ever lost, you just click Reset again and get a new one.

Part 1 — Switch the button on (do this ONCE)

Heads up: this one-time step uses a few computer commands. If you're not comfortable, hand this page to whoever looks after your website / Firebase — it takes about 5 minutes. After this, nobody ever needs to touch it again.

1 Open the black command window

  • On the computer that has the app's project folder, open Command Prompt (click Start, type cmd, press Enter).
  • Go to the project folder. If it's on the C drive, type this and press Enter:
Type this, then press Entercd C:\Users\John.JASREGALADO\.claude\BASIC_by_JMSI

2 Install the two helpers

Type each line, press Enter, and wait for it to finish before the next one.

Line 1npm install -g firebase-tools
Line 2npm --prefix password-reset-kit/functions install

3 Log in to Firebase

Type this, then press Enterfirebase login
  • A web browser opens. Sign in with the Google account that owns your Firebase (your owner email).
  • Click Allow. When it says you're signed in, close the browser tab.

4 Turn on the button

Type this, then press Enterfirebase deploy --only functions:adminResetPassword
  • Wait until you see "Deploy complete!" (about a minute).
  • That's it — the 🎲 Reset button now works. You never do Part 1 again.
If it says something about a "Blaze plan," your Firebase must be on the pay-as-you-go plan — which you already turned on earlier. If not, turn it on in the Firebase website, then run the command again.

Part 2 — How to use it every day

This is the easy part. Anyone can do it.

1 Open the worker list

In the app, open Accounts & Roles (from the menu). You'll see all your workers.

2 Click Reset next to the worker

  • Find the worker's name. Click the 🎲 Reset button on their row.
  • A box asks "Reset this password?" — click Reset to random.

3 Copy the new password

  • The app shows the new password one time. Click Copy.
  • ⚠️ It will not be shown again — but that's okay, you can always reset again.

4 Save it on the shop device

  • Go to the shop iPad or computer the worker uses.
  • Open the app, type the worker's email and paste the new password, and sign in.
  • When the browser asks "Save password?", tap Save. ✅
  • Now the worker just taps Sign in — it fills itself. They never see the password.

When to use it

New workerReset to make their first password, then save it on the shop device.
Worker leavesReset their password (their saved login stops working), or switch their account off.
Device lost or stolenReset the passwords of everyone whose login was saved on it.
That's the whole thing. Part 1 once, Part 2 whenever you need. No passwords to write down, nothing to remember.
