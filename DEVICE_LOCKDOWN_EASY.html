Locking the app to shop devices — the easy way

Follow each step in order. If you can use a phone, you can do this. Do the whole guide once per shop iPad and once per shop computer.

What this does (in plain words)

Each worker gets a secret password they never see. You type it into the shop's iPad or computer one time, and the device remembers it. After that, the worker just taps Sign in and they're in — without ever knowing the password.

So at the shop, work is easy. But at home, on their own phone, they don't know the password, so they can't get in. That's the whole trick.

You (the owner/admin) are different. You know your own password, so you can still open the app from anywhere. This guide only locks down the workers.
Be honest with yourself: this stops normal, honest staff. A very sneaky person on a Windows computer could still peek at the password. That's why the iPad is the safest — do the important stations on iPads.

Part A — Make the secret password (do this once per worker)

1 Make each worker's login with a long, random password

  • When you add a worker's account, give them a long, silly, random password. Example: Bakal-7739-Lobo-Kalye.
  • Write it in your own private notebook (or a locked notes app on your phone). One line per worker.
  • Never tell the worker the password. Never write it on the device or on a sticky note.
Tip: make it easy to type once but impossible to guess — mix words and numbers, like Motor-4182-Pusa. You only type it a few times total.

Part B — Set up a shop iPad SAFEST

Do this on each shop iPad. Takes about 10 minutes.

2 (If the iPad was used before) Erase it clean

  1. Open Settings → General → Transfer or Reset iPad → Erase All Content and Settings.
  2. Let it restart. (Skip this step if the iPad is brand new.)

3 Give it a passcode only YOU know

  1. During setup (or in Settings → Face ID & Passcode), set a passcode.
  2. This passcode is the manager's secret. Write it in your notebook. Do not give it to workers.
Why: to peek at a saved password, the iPad asks for this passcode. If workers don't know it, they can't peek.

4 Do NOT sign in with a personal Apple ID

  1. When the iPad asks to Sign in with your Apple ID / iCloud, tap "Set Up Later in Settings" or Skip.
  2. If it's already signed into someone's Apple ID: Settings → tap the name at the top → Sign Out.
Why: this stops the saved password from copying itself onto anyone's personal iPhone.

5 Log in once and let Safari save it

  1. Open Safari. Type the shop's app link, for Fairview:
    https://basicautomotiveservices.com/fairview
    (Use that branch's own link for other branches.)
  2. Type the worker's email and the secret password from your notebook. Tap Sign in.
  3. Safari asks "Save Password?" → tap Save Password. ✅

6 Make an app icon on the Home Screen

  1. In Safari, tap the Share button (the square with an arrow pointing up).
  2. Tap Add to Home Screen → name it BASIC → tap Add.
  3. Now there's a BASIC icon like a real app. Workers tap that, then tap Sign in — it fills the password by itself.

7 Lock the iPad to just this app (Guided Access)

  1. Go to Settings → Accessibility → Guided Access → turn it ON.
  2. Tap Passcode Settings → Set Guided Access Passcode → set a passcode (manager's secret — can be the same as step 3).
  3. Open the BASIC app. Triple-click the top button (the power button) → tap Start.
  4. Now the iPad is stuck on the app. To get out, triple-click again and type your passcode.
Done! This iPad now only shows the app, workers can't peek at the password, and the password can't jump to their phone.

Part C — Set up a Windows shop computer

Do this on each shop computer. Takes about 10 minutes.

8 Make a "Shop" account the workers will use

  1. Open Settings → Accounts → Other users → Add account.
  2. Click "I don't have this person's sign-in information" → "Add a user without a Microsoft account."
  3. Name it Shop. Set a password and keep it secret (write it in your notebook).
  4. Sign out and sign in to the Shop account.

9 Open Chrome and DON'T sign in to Google

  1. Open Google Chrome.
  2. If it asks you to sign in / turn on sync, click "No thanks" or close it.
  3. Check: Chrome menu (three dots ⋮) → Settings → the top should say you're not signed in. Good.
Why: if Chrome signs into a Google account, the saved password copies to any phone with that account. We don't want that.

10 Log in once and let Chrome save it

  1. Go to the shop's app link (e.g. https://basicautomotiveservices.com/fairview).
  2. Type the worker's email and the secret password. Click Sign in.
  3. Chrome asks "Save password?" → click Save. ✅

11 Make it open by itself

  1. Chrome menu (⋮) → Settings → On startup → choose "Open a specific page or set of pages" → Add a new page → paste the app link.
  2. Right-click the Chrome icon on the taskbar → Pin to taskbar.
  3. Tell workers: open Chrome, click Sign in — it fills the password by itself.
Windows is a little weaker than iPad. A very curious person could still dig out the password on a computer. If a station really matters, use an iPad instead.

Everyday rules (keep it working)

When this happens…Do this
New worker joinsMake their login with a new random password (Part A), then log in once on the shop devices and tap Save (steps 5 & 10).
Worker leaves / is firedIn the app's Accounts page, switch their account off (or change their password). The saved login stops working right away.
iPad or computer is lost / stolenChange the passwords of every worker whose login was saved on that device (or switch those accounts off). Now the saved logins are useless.
You (owner/admin)You know your own password — you can log in from anywhere, anytime. Nothing changes for you.

Quick checklist per device

iPad:

  • Manager-only passcode set
  • NOT signed into a personal Apple ID
  • Logged in once, password Saved
  • BASIC icon added to Home Screen
  • Guided Access turned on and Started

Windows computer:

  • Separate "Shop" account, password kept secret
  • Chrome NOT signed into Google (sync off)
  • Logged in once, password Saved
  • App link set to open on startup, Chrome pinned
That's it. Workers can do everything at the shop with one tap, and can't log in from home because they never learned the password. Repeat this guide for every new branch and every new device.
